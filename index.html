Monitorización y alertado

Miguel Ángel García Martínez

¿Qué monitorizar/alertar?

  • Sistema
  • Programa en ejecución
  • Tendencias
  • Aplicación
  • Logging

Ventajas

  • Obtener datos de lo monitorizado
  • Comparar con datos anteriores
  • Comparar con otras máquinas/programas
  • Previsión de recursos
  • Ahorro de costes
  • Detección precoz de problemas

Programas

  • Nagios, Pandora: Sistema
  • Zabbix: Sistema (monitorización)
  • Munin: Sistema
  • New Relic: SaaS (pago)
  • Graphite: Monitorización
  • Sensu: Alertados
  • Grafana: IDE
  • FlapJack: Enrutado de alertas
  • PagerDuty: Enrutado de alertas (SaaS)
  • Sentry: Agrupador de logs
  • RSyslog/Logstash/ElasticSearch: Agrupación de logs de sistema

Hoy

  • Munin
  • Graphite
  • Grafana
  • Sensu
  • Sentry

Munin

  • Perl
  • Sencillo
  • RRD
  • Estático en /var/cache/munin/www
# apt-get install munin munin-node
# munin-node-configure --suggest --remove-also --shell | bash

Graphite

  • Python/Django
  • API web
  • Whisper (evolución de RRD
  • Soporta eventos discretos
# apt-get install graphite-web

Grafana

  • IDE para Graphite
  • Muy muy molón y configurable

Sensu

  • Solo alertas
  • Pero puede incluir gráficas de Graphite/Grafana

Sentry

  • API "Raven"
  • Ideal para configurar un handler del logger.
